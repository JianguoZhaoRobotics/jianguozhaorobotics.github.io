## Course Description

Advanced topics in mechanical systems analysis, modeling, and design. Topics include differential equations, control systems, trajectory optimization, and reinforcement learning applications to robotic systems. 

This course combines theory with hands-on programming labs using Python and Jupyter notebooks. Students work through 10 comprehensive labs plus a capstone project on acrobot control.
